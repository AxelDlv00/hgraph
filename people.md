# People

These worked examples are maintained as part of **hgraph**. This tab is itself a
demonstration: it is a plain Markdown file wired in through the workspace
`config.yaml`'s `tabs:` list — no code change required.

## Maintainers

- **Axel Delaval** — author of hgraph and these examples.

## Contributing

hgraph is a plain-files graph: every node and edge is a Markdown file with YAML
metadata, so contributing a statement, a review, or a comment is an ordinary Git
change. See the project [README](https://github.com/AxelDlv00/hgraph) to get
started.

*Add your own tabs — People, Roadmap, Acknowledgements — by dropping a `.md` or
`.html` file next to this one and listing it under `tabs:`.*
